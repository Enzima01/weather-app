# Weather App 🌤️

Aplicação em Java que consulta dados de clima utilizando uma API online.

## Versões

* **CLI** – versão executada no terminal
* **GUI** – versão com interface gráfica (Java Swing)

## Tecnologias

* Java 21
* HTTP Client API
* JSON (org.json)
* WeatherAPI

## Estrutura do Projeto

```
weather-app
 ├── cli
 │   └── versão de linha de comando
 └── gui
     └── versão com interface gráfica
```

## API utilizada

https://www.weatherapi.com/
