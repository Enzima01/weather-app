# Weather App 🌤️

Aplicação em Java que consulta dados de clima utilizando uma API online

## Versões

* **CLI** – versão executada no terminal
* **GUI** – versão com interface gráfica (Java Swing)

## Tecnologias

* Java 21
* HTTP Client API
* JSON (org.json)
* WeatherAPI

## Execução

O projeto gráfico pode ser executado via arquivo `.jar` no terminal

#### GUI
```bash
java -jar WeatherApp.jar
```
